GIS

For the final project of my GIS course, me and my team were asked to make a geographical information system about an issue that is currently a problem. We decided to analyze where are the suitable areas to build a bus stop in DKI Jakarta. In this project we used ArGIS Pro and ArcGIS Online. We obtained the data from the enrich tool in ArgisPro. Then, we compare the data by using different colors for each layer so they can show the area that need more bus stop. The dashboard can be viewed online, however, user needs to login with an Esri account
